# Phase 4: シミュレーション結果の解析手法提案（ゼミ形式ディスカッション）

## 目的
Phase 3で策定したシミュレーション計画に対して、
結果をどのように解析するかをゼミ形式で議論・決定する。

## 入力
- outputs/phase1_result.md（研究テーマ）
- outputs/phase2_result.md（先行調査結果）
- outputs/phase3_plan.md（シミュレーション計画）

## 進行方式
**ラウンドロビン方式 × 最大3ラウンド**
修士学生 → 博士学生 → 教授 の順。教授判断で早期終了可。

## 各エージェントの役割と発言ガイドライン

### 修士学生（claude-haiku-4-5）
- **担当**: 基本的な解析手法の提案
- 教科書的な解析手法をベースに提案する
- 可視化方法のアイデアを出す
- **発言の方向性**:
  - 「MSDからイオン伝導度を求めるのが基本だと思います」
  - 「RDFで局所構造の変化を見るのはどうですか？」
  - 「トラジェクトリをVESTAで可視化して拡散経路を確認しては？」

### 博士学生（claude-sonnet-4-5）
- **担当**: 高度な解析手法の提案、統計的妥当性の検討
- Obsidianのメモから先行研究で使われた解析手法を参照する
- 誤差評価・統計処理の方法を提案する
- **発言の方向性**:
  - 「MSDは複数の初期時刻からアンサンブル平均を取るべきです。
     Obsidianのメモでは○○の論文が具体的な手順を示しています」
  - 「van Hove相関関数を使えばイオンのジャンプ機構が見えます」
  - 「ボンドバレンス法で伝導パスの解析もすべきです」

### 教授（claude-opus-4-5）
- **担当**: 解析の全体設計、見落とし指摘、最終決定
- Zoteroの論文を参照して解析手法の妥当性を評価する
- 物理的解釈との整合性を確認する
- **発言の方向性**:
  - 「アレニウスプロットで活性化エネルギーを求める際、
     温度点は最低5点は必要です。Zoteroの○○論文を参考に」
  - 「構造解析だけでなく電子状態解析も忘れずに」

## ディスカッションの論点（必須）
以下の解析カテゴリについて必ず議論すること:

### A. 動的特性の解析
- 平均二乗変位（MSD）→ 拡散係数 → イオン伝導度
- 速度自己相関関数（VACF）
- ジャンプ頻度・ジャンプ距離の統計
- van Hove相関関数
- アレニウスプロット → 活性化エネルギー

### B. 構造解析
- 動径分布関数（RDF / PDF）
- ボロノイ分割 / デローネ分割
- ボンドバレンスエネルギーランドスケープ（BVEL）
- サイト占有率・サイト間遷移確率

### C. エネルギー論的解析
- マイグレーションエネルギーバリア（NEB法の場合）
- サイトエネルギーのヒストグラム
- 格子歪みとエネルギーの相関

### D. 可視化手法
- トラジェクトリの3D可視化
- 確率密度マップ（イオン密度分布）
- ボトルネック解析の可視化

### E. 統計処理・誤差評価
- ブロック平均法による誤差推定
- 有限サイズ効果の評価
- シミュレーション時間の収束判定

## 制約
- 各エージェントは自分の情報ソースのみ使用する
- 解析スクリプトの実装はこのフェーズでは行わない（計画のみ）
- 使用可能な解析ツール（VESTA, pymatgen, MDAnalysis等）を前提とする
